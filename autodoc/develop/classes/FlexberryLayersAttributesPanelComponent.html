    L.Popup private

    Defined in addon/components/flexberry-layers-intersections-panel.js:391

    Finishes identification.

    Parameters:

    • e Object

      Event object.

      • latlng L.LatLng

        Center of the polygon layer.

    • options.polygonLayer L.Polygon

      Polygon layer related to given vertices.

    • excludedLayers Object[]

      Objects describing those layers which were excluded from identification.

    • layers Object[]

      Objects describing those layers which are identified.

    • results Object[]

      Objects describing identification results. Every result-object has the following structure: { layer: ..., features: [...] }, where 'layer' is metadata of layer related to identification result, features is array containing (GeoJSON feature-objects)[http://geojson.org/geojson-spec.html#feature-objects].

    Returns:

    L.Popup:

    Popup containing identification results.
