    _identify

    (
    • e
    • layers
    • results
    )
    private

    Inherited from BaseLayerComponent: addon/components/base-layer.js:562

    Handles 'flexberry-map:identify' event of leaflet map.

    Parameters:

    • e Object
      Event object.
      • boundingBox L.Rectangle

        Leaflet layer representing bounding box within which layer's objects must be identified.

      • latlng L.LatLng

        Center of the bounding box.

    • layers Object[]
      Objects describing those layers which must be identified.
    • results Object[]
      Objects describing identification results. Every result-object has the following structure: { layer: ..., features: [...] }, where 'layer' is metadata of layer related to identification result, features is array containing (GeoJSON feature-objects)[http://geojson.org/geojson-spec.html#feature-objects] or a promise returning such array.
